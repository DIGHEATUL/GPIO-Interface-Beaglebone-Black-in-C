# GPIO-Interface-Beaglebone-Black-in-C

There Are some Important Simple Function that we used in Beaglebone Black Gpio.
1) set_gpio_high :
   Sets the Gpio High.

2) set_gpio_low :
   Sets the Gpio low.

3) read_gpio :
   reads the Value of Gpio in terms of 0 and 1.

Required sudo permission for Run.
